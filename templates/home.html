About the Video Game Dataset

The data used as part of this project is from the CORGIS Datasets Project. For this webapp I am using the Video Games dataset from that website, which can be found here.

Included in this dataset is information about games released from 2004 until 2008. That information includes the name, publisher, rating, sales and, if applicable, average playtime of each individual game. Playtime is listed as hours for the category which is chosen (average playtime, average completion playtime, average main story playtime, etc.)

Two things to note: First, if the playtime of any game is 0 that section of data was not polled for (i.e. the "polled" variable for that section also equals zero). Second, sales are recorded in millions of dollars, so they will appear as decimal numbers ("$0.41 million" or "$1.5 million" instead of "$140,000").
